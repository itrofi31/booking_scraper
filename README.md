## ⚡ Быстрый старт (5 минут)

### 1. Установка зависимостей

```bash
pip install playwright openpyxl
playwright install chromium
```

### 2. Запуск — одна дата
```bash
python scraper.py --checkin 2025-07-15 --checkout 2025-07-16 --adults 2
```

### 3. Запуск — пакет дат (рекомендуется)
```bash
python scraper.py --dates-file dates.txt --output результаты_июль.xlsx
```

### 4. Только мои объекты (быстро, без поиска конкурентов)
```bash
python scraper.py --dates-file dates.txt --no-competitors
```

## ⚠️ Важные ограничения и риски

### Блокировки Booking.com
- Booking.com активно блокирует ботов. Скрипт использует:
  - Реалистичный User-Agent (ротация)
  - Случайные задержки между запросами (1.8–4.5 сек)
  - Маскировку webdriver
- **Если получаешь 403 или captcha:** подожди 30–60 мин, или запускай с VPN/другого IP
- **Рекомендация:** не запускать чаще 1–2 раз в день

### Структура страниц меняется
- Booking.com регулярно обновляет HTML/CSS
- Если парсер перестал работать — нужно обновить CSS-селекторы в функции `scrape_property()`
- Проверяй `scraper.log` — там детали каждого запроса

### Валюта и цены
- Цены возвращаются в THB (тайский бат)
- Убедись, что на Booking.com у тебя выбрана валюта THB (параметр `selected_currency=THB`)
- Цены могут включать/не включать налоги в зависимости от настроек страницы

### Количество гостей
- **Важно:** цена сильно зависит от `--adults`
- Для корректного сравнения с конкурентами используй одинаковое число гостей для всех
- Стандарт для анализа: 2 взрослых

## 🔧 Если что-то не работает

### Ошибка: "no_availability_table"
Booking не показывает цены — либо:
- Объект заблокирован на выбранные даты
- Страница не загрузилась полностью
- Нужен авторизованный сеанс

**Решение:** увеличь `PAGE_LOAD_WAIT = 5.0` в начале скрипта

### Ошибка: "no_offers_parsed"
Страница загрузилась, но таблица комнат не найдена.

**Решение:** запусти скрипт с `headless=False` для отладки:
```python
# В функции new_context() измени:
browser = await playwright.chromium.launch(headless=False, ...)
```
Это откроет видимое окно браузера — посмотришь что происходит.

### Playwright не установлен
```bash
pip install playwright
playwright install chromium
# На Linux может потребоваться:
playwright install-deps chromium
```

## 📊 Формат вывода Excel

Файл `booking_analysis.xlsx` содержит:
- Блоки по датам (каждая пара заезд/выезд — отдельный блок)
- Мои объекты выделены зелёным цветом (★)
- Колонка **"vs Мой объект %"**: зелёный = конкурент дороже (хорошо), красный = дешевле
- Колонки: Объект, Тип комнаты, Мин/Макс цена, Отмена, Завтрак, Рейтинг, Отзывы

## 💡 Советы по использованию

1. **Запускай раз в неделю** на ключевые даты (праздники, пиковые периоды)
2. **Сохраняй файлы с датой** в названии: `анализ_2025-07-15.xlsx`
3. **Используй `--no-competitors`** если хочешь быстро проверить только свои цены
4. **Редактируй `dates.txt`** под актуальные даты — добавляй важные события (Songkran, НГ и т.д.)
